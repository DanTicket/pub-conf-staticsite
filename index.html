What is a Hash Function?

A hash function takes an input (or 'message') and returns a fixed-size string of bytes. The output is typically a seemingly random sequence of characters, known as the hash or digest. Hash functions are widely used in security, digital signatures, message authentication, integrity checks, blockchain, cryptocurrency, storing/validating passwords and many other secure applications.

Deterministic
The same input always produces the same output hash.
Fixed Output Size
No matter the input size, the hash output length is always constant.
Fast Computation
Hash functions can quickly generate hashes for any input.
Preimage Resistance
It should be infeasible to reconstruct the input from its hash.
Collision Resistance
It is very unlikely two different inputs produce the same hash.
Avalanche Effect
Small changes in input produce vastly different hashes.

Key Quantum Principles

There are a handful of very famous and important principles you should know about any state that is in a quantum 'superposition'. Refer back to these as you need to help you follow along!

Superpostion
A qubit can be in a combination of both |0‚ü© and |1‚ü© states at the same time. This allows quantum systems to represent many possible outcomes simultaneously, unlike classical bits which are either 0 or 1.
Mesuring a State Causes Collapse
Taking a measurement of (looking at) a quantum state causes it to immediately collapse into a basis state. It is no longer representing many things at once. The outcome is probabilistic, not deterministic.
Entanglement
Qubits can become entangled, meaning the state of one qubit cannot be described independently of the other. Measuring one instantly affects the state of the other ‚Äî even across distance. We can use this to teleport information (yes, really) and defend against man-in-the-middle attacks.
Interference
Quantum amplitudes can interfere constructively or destructively. Algorithms use this to amplify correct results and cancel out wrong ones. This is what gives quantum algorithms their power, e.g., Grover's search.
